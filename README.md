# x86 Assembly Projects

This repository contains several projects that demonstrate my experience with x86 assembly language, low-level programming, and systems programming in C. These projects include inline assembly exercises, assembly-based function implementations, and a multithreaded prime number generator using pthreads.

## Projects Included

### Assembly Fundamentals Exercises
A collection of small programs focused on core assembly concepts such as register usage, control flow, memory access, function calls, and inline assembly.

### Assembly Function Implementations
A project centered on implementing required functions in assembly and integrating them with C test code.

### Multithreaded Prime Number Generator
A C program that uses pthreads to generate prime numbers across multiple threads, showing concurrency and performance-oriented programming concepts.

## Skills Demonstrated
- x86 assembly language
- Inline assembly in C
- Low-level memory and register operations
- Function calling conventions
- Systems programming
- Multithreading with pthreads
